# mac 自带apache


## 启动Apache

    sudo apachectl -v

查看apache版本;

    sudo apachectl start
    sudo apachectl restart
    sudo apachectl stop

apache控制命令;

1. 默认apache路径在“/Library（资源库）/WebServer/Documents/”下面;
2. Apache的安装目录在：/etc/apache2/, 命令 `open /etc`查看
