# 关于Python脚本开头两行

	#!/usr/bin/python和
	# -*- coding: utf-8 -*-

1. `#!/usr/bin/python`

	是用来说明脚本语言是python的

	要用/usr/bin下面的程序（工具）python，这个解释器，来解释python脚本，来运行python脚本的。

2. `# -*- coding: utf-8 -*-`

	是用来指定文件编码为utf-8的
